# 🔐 Syntecxhub Password Manager

A secure and beginner-friendly Password Manager built using Python and Cryptography library.  
This project securely stores user credentials in encrypted format using symmetric encryption and master password authentication.

---

## ✨ Features

✅ Master Password Authentication  
✅ Encrypted Local Storage  
✅ Add Password Entries  
✅ Retrieve Saved Passwords  
✅ Search Credentials  
✅ Delete Credentials  
✅ Secure Vault File (`vault.dat`)  
✅ Beginner-Friendly CLI Interface  

---

## 🛠️ Technologies Used

- 🐍 Python
- 🔒 Cryptography Library
- 📂 JSON Storage
- 💻 VS Code

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Syntecxhub_Password_Manager.git
```

Move into project folder:

```bash
cd Syntecxhub_Password_Manager
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Project

```bash
python password_manager.py
```

---

## 🔐 Security Implementation

This Password Manager uses:

- PBKDF2 key derivation
- Symmetric encryption
- Encrypted local vault storage
- Master password-based authentication

All passwords are encrypted before saving to disk.
